1. What is Python?

Python is a high-level programming language known for its simplicity and readability.

2. What are some features of Python?

Python features include dynamic typing, automatic memory management, and extensive standard libraries.

3. How do you write a comment in Python?

In Python, comments start with the '#' symbol and extend to the end of the line.

4. What is the purpose of indentation in Python?

Indentation is used to define the structure and scope of code blocks in Python, such as loops, conditionals, and functions.

5. How do you declare a variable in Python?

In Python, variables are declared by simply assigning a value to a name, e.g., 'x = 10'.

6. What are the different data types supported by Python?

Python supports data types such as integers, floats, strings, lists, tuples, dictionaries, and booleans.

7. How do you print output in Python?

You can print output in Python using the 'print()' function, e.g., 'print("Hello, world!")'.

8. What is a list in Python?

A list in Python is an ordered collection of elements, which can be of different data types, enclosed within square brackets.

9. How do you define a function in Python?

You can define a function in Python using the 'def' keyword followed by the function name and parameters, e.g., 'def my_function(x, y):'.

10. What is a loop in Python?

A loop in Python is used to execute a block of code repeatedly. Python supports 'for' and 'while' loops.

11. How do you comment multiple lines in Python?

In Python, you can comment multiple lines using triple quotes (''' ''').

12. What is the purpose of the 'if' statement in Python?

The 'if' statement is used for conditional execution in Python. It allows you to execute a block of code only if a certain condition is true.

13. What is a dictionary in Python?

A dictionary in Python is an unordered collection of key-value pairs, enclosed within curly braces.

14. How do you check the length of a list in Python?

You can check the length of a list in Python using the 'len()' function, e.g., 'len(my_list)'.

15. What is the purpose of the 'else' statement in Python?

The 'else' statement is used in conjunction with 'if' statement to execute a block of code when the condition is false.

16. What is a tuple in Python?

A tuple in Python is an ordered collection of elements, which are immutable and enclosed within parentheses.

17. How do you access elements in a list in Python?

You can access elements in a list in Python using index notation, e.g., 'my_list[0]'.

18. What is the purpose of the 'for' loop in Python?

The 'for' loop in Python is used to iterate over a sequence (such as a list, tuple, or string) and execute a block of code for each element in the sequence.

19. How do you concatenate strings in Python?

You can concatenate strings in Python using the '+' operator, e.g., 'str1 + str2'.

20. What is the purpose of the 'range()' function in Python?

The 'range()' function in Python is used to generate a sequence of numbers. It can take one, two, or three arguments to specify the start, end, and step size of the sequence.
