Python Viva Practice Questions

1. What is Python?

Python is a high-level, interpreted, and general-purpose programming language known for its simplicity and readability. It was created by Guido van Rossum and first released in 1991.

2. What are the key features of Python?

  • Easy to Learn and Read
  • Interpreted Language
  • High-level Language
  • Object-Oriented
  • Extensive Standard Library
  • Dynamic Typing

3. How does Python handle indentation?

Python uses indentation to define blocks of code, such as loops, functions, and classes. Proper indentation is crucial for readability and correct program execution.

4. What is dynamic typing in Python?

Dynamic typing in Python means that you don't need to explicitly declare the variable type. Variable types are inferred at runtime based on the assigned value.

5. Explain the difference between tuples and lists in Python.

Tuples are ordered collections of items that are immutable, meaning they cannot be changed after creation. Lists, on the other hand, are mutable, meaning their elements can be modified.

6. How are comments denoted in Python?

Comments in Python start with the # symbol and are used to provide explanations or annotate code. Comments are ignored by the Python interpreter during execution.

7. What are the basic data types in Python?

  • Integers
  • Floats
  • Strings
  • Lists
  • Tuples
  • Dictionaries
  • Sets

8. How do you define a function in Python?

Functions in Python are defined using the def keyword followed by the function name and parameters. For example: def my_function(param1, param2):

9. What is the purpose of the if statement in Python?

The if statement is used for conditional execution. It allows you to execute certain code blocks based on whether a given condition is true or false.

10. How do you open and read from a file in Python?

You can open and read from a file in Python using the open() function with the appropriate file mode (e.g., "r" for reading). For example: with open("file.txt", "r") as f:

11. Explain the concept of inheritance in Python.

Inheritance is a feature of object-oriented programming in Python that allows a class to inherit attributes and methods from another class. It facilitates code reusability and promotes the DRY (Don't Repeat Yourself) principle.

12. What is the purpose of the __init__ method in Python classes?

The __init__ method is a special method in Python classes used for initializing object instances. It is called automatically when a new object is created.

13. How do you handle exceptions in Python?

Exceptions in Python are handled using try, except, and optionally finally blocks. Code that may raise an exception is placed within the try block, and corresponding exception handling code is placed within the except block.

14. Explain the concept of polymorphism in Python.

Polymorphism is the ability of different objects to respond to the same method call in different ways. In Python, polymorphism is achieved through method overriding and method overloading.

15. What is the purpose of the range() function in Python?

The range() function in Python generates a sequence of numbers within a specified range. It is commonly used in for loops to iterate over a sequence of numbers.
